## Data Type
- Byte
- Short
- Int
- Long
### Byte
- 1 byte = 8 bits
- 범위 : -2^7 ~ 2^7 - 1
### Short
- 1 short = 2 bytes
- 범위 : -2^15 ~ 2^15 - 1
### Int
- 1 int = 4 bytes
- 범위 : -2^31 ~ 2^31 - 1
### Long
- 1 long = 8 bytes
- 범위 : -2^63 ~ 2^63 - 1
